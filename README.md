## 💻 Local Development

Want to run this portfolio locally? It's lightweight and easy to set up.

### Prerequisites

- Node.js (v18+ recommended)
- npm or yarn

### Installation

1. **Clone the repository:**

   ```bash
   git clone [https://github.com/ZaikOSS/portfolio.git](https://github.com/ZaikOSS/portfolio.git)
   cd portfolio

   ```

2. **Install dependencies:**

```bash
npm install

```

3. **Start the development server:**

```bash
npm run dev

```

4. **Open your browser:**
   Navigate to `http://localhost:8080` (or the port provided in your terminal).
